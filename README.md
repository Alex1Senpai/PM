## Day 1 — что прочитать в первую очередь

Если ты новый разработчик в команде, прочитай эти файлы в этом порядке. Полный onboarding ~30 минут.

### Шаг 1 (5 мин) — этот файл
README.md (тот, что ты читаешь). Узнаёшь общий контекст: воронка, кто что делает, как мы коммитим и мержим.

### Шаг 2 (5 мин) — обзор продукта
`Funnel/3-Parent-App/_Hub.md` — что строит команда сейчас, какой дедлайн, какой scope, какие архитектурные решения уже приняты.

### Шаг 3 (5 мин) — end-state
`Funnel/3-Parent-App/end-state-v1.md` — три сценария того, что родитель должен мочь делать в приложении 15 августа. Это твой North Star для UX-решений.

### Шаг 4 (10 мин) — твоя задача
Найди файл задачи со своим именем в поле `assignee:`:
- Parent App: `Funnel/3-Parent-App/Tasks/`
- Сайт: `Funnel/1-School-site/Tasks/`

См. полный README в репозитории для деталей (структура задачи, Definition of Done, Decisions). Далее идёт объёмный onboarding-блок — пропускается в этом сокращённом push'е ради размера сообщения.

См. полный README в локальном репозитории (с Claude-agent инструкциями).
