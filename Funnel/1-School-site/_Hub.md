# School-site — Верх воронки

## Что это
Маркетинговый сайт Оксбриджа. Первая точка контакта для почти каждого потенциального родителя. Задача: быть найденным в поиске, ясно рассказать о школе, захватить квалифицированных лидов в CRM.

## Где находится в воронке
- **Стадия:** Верх
- **Откуда приходят:** поисковики, платная реклама, сарафанное радио, соцсети
- **Куда уходят:** Захват лида → [[../2-CRM/_Hub]]

## Код
Код сайта живёт в репозитории **`oxbridge-school-site`**. Полная карта код-репозиториев и локальных клонов — в [[../../code-repos]].

## Текущее состояние
- **Статус:** активен, поддержка + работа по CWV
- **Владелец:** Bekhruz
- **Жёсткий дедлайн:** нет — непрерывное улучшение
- **Мягкая цель:** регрессии CWV на главной + tuition-fees страницах закрыты до следующего цикла приёма

## Видение (длинная тень)
Сайт растёт в более насыщенный опыт для потенциальных родителей: туры по кампусу, виртуальные мероприятия, глубокие материалы по программам, прозрачный процесс приёма. В конечном итоге интегрируется с CRM, чтобы потенциальные родители видели персонализированный контент по стадии.
